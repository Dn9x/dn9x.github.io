上一篇记录了属性选择器，在CSS中选择器有很多中，元素选择器，类选择器，ID选择器，后代选择器，子元素选择器，相邻选择器，伪类选择器，伪元素选择器。这里主要记录的都是CSS3中新增的东西，当然之前的我也会简单的记录下。 ------------------------------------ ###元素选择器 元素选择器很好理解，就是直接根据html中的标签匹配，比如： ``` h1{ color:red; } ``` 这样整个html页面中的所有h1标签的字体颜色都是红色。 ---------------------------------------------------------- ###类选择器 类选择器就是样式里面通过 ‘.‘ + 样式名称 来定义样式，html中通过`class`来使用样式。比如： ``` css .title{ color:red; } html <h1 class='title'>这里是标题</h1> ``` 类选择器是使用最多的一种选择器，灵活，方便是优势。*这里的样式名称是随便定义的，这里这样提是为了后面介绍另外一种选择器。* ---------------------------------------------------- ###ID选择器 ID选择器使用和类选择器一样只是通过 ’#‘ + 样式名称 定义样式，html中国`id`来匹配，比如： ``` css #title{ color:red; } html <h1 id='title'>这里是标题</h1> ``` ID选择器优点就是可以保持唯一，缺点也是只能保持唯一，因为在html中id不能重复，所以ID选择器的样式也就只能一个元素使用了。 -------------------------------------------------- ###后代选择器 后代选择器就是指某个元素内部的匹配元素不管是第几代(第多少层)，这里很像子元素选择器不过是不同的，后面会提到。后代选择通过空格分开两个元素就行了。比如： ``` css h1 b { color:red; } html <h1>这里是<b>后代选择</b> 器</h1> <p>这里是<b>后代选择</b> 器</p> ``` 这里的效果就是只有h1里面的b标签的字体颜色会变成红色。p标签里面的b标签字体是不会变的。效果如： ![](../../image/2014-03-27css3-1.JPG) ----------------------------------------------- ###子元素选择器 子元素选择器只会匹配元素里面第一层级的元素，第二层级的就不再匹配，直接看效果： ``` css h1 b { color:blue; } h1 > b { color:red; } html <h1>这里是<b>子元素选择</b> 器</h1> <h1>这里是<i><b>后代选择</b></i> 器</h1> ``` 上面这样或许就很清楚了，效果如下： ![](../../image/2014-03-27css3-2.JPG) --------------------------------------------------- ###相邻选择器 相邻选择器会匹配在同一层级的两个相邻元素的后面一个上面，相邻选择器通过’+‘连接，比如： ``` css h1 + b { color:blue; } html <h1>前奏</h1> 这里是<i><b>后代选择</b></i> 器 <h1>前奏</h1> 这里<b>是后代选</b>择器 ``` 这里h1标签的父标签是body。效果如下： ![](../../image/2014-03-27css3-3.JPG) 这里可以看出两个标签必须是在同一层级才会匹配，不再同一层级是不会匹配的。 --------------------------------------------- ###伪类选择器 伪类选择器可以理解和类选择器是相反的，但是又不相反的。相反的是类选择器名称可以随便定义，但是伪类选择器的名称不能随便定义。名称是定义好的我们直接使用。不同的是使用方式不同。最常用的伪类选择器就是a标签的伪类选择器了。如： ``` a:link a:visited a:hover a:active ``` ---------------------------------------------------- ---------------------------------------------------- css中的选择器如果灵活使用会可以减少很多的编码时间，比如如果多个元素有一些公用的属性可以这样： ``` css .title, .box, .cell{ color: red; } ``` 这样就可以避免三个样式都分别写三次颜色值。比如还可以同一个样式只是匹配某种元素标签可以这样： ``` css p.title{ color:red; } ``` 这样就可以匹配只有在p标签上使用title样式才会有效果。当然方式是有很多很多的只要自己会灵活使用就会很方便。 -------------------------------------------------- 上面主要记录的都是在CSS中基本的选择器，CSS3中都还是正常使用的。而伪元素选择器没有介绍是因为会放到下节的结构性伪类选择器中记录。下一章会记录CSS3新增的结构性伪类选择器。
