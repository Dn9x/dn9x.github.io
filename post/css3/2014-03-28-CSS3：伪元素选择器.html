前面记录了属性选择器和基本选择器，这一节主要记录伪元素选择器，伪元素选择器顾名思义就是不是真正元素使用的选择器，伪元素选择器都是定义好的，我们直接使用就行了。伪元素选择器的使用格式如下： ``` 选择器：伪元素 { 属性：值 } ``` 这里的选择器就是前面提到的各种选择器，随便使用哪种都行。伪元素选择器总共就只有四个，所以记住还是很方便的，而且伪元素选择器最常用的还只有两个而已`before`, `after`。 ----------------------------------------------------- ###first-line选择器 first-line选择器可以作用于某元素中第一行文字使用样式，而且只会作用于第一行。比如： ``` css div{ width:200px; height: 200px; border: 1px solid #ccc; } div:first-line{ color: red; } html <div> 这里演示的是第一个div中的内容前面记录了属性选择器和基本选择器，这一节主要记录伪元素选择器，伪元素选择器顾名思义就是不是真正元素使用的选择器，伪元素选择器都是定义好的，我们直接使用就行了。伪元素选择器的使用格式如下 </div> ``` 这里注意啊，我div中的内容我没有换行，是浏览器自动帮我换行的，我们看下效果如下： ![](../../image/2014-03-28css3-1.jpg) 这里可以看出只会改变第一行的样式。 --------------------------------------------------- ###first-letter选择器 first-letter选择器有时候还是很方便的，它的作用是改变某元素里面的第一个文字的样式。比如修改刚才的样式如下： ``` css div:first-letter{ font-size: 34px; color: blue; } ``` 效果如下： ![](../../image/2014-03-28css3-2.jpg) 这里只会改变第一个字的样式，如果有多个div那么就会改变多个div中的第一个字。 --------------------------------------- ###before选择器 before选择器通常都是和content属性搭配，因为他可以在某个元素之前插入内容。内容可以是文字或者图片都行。比如： ``` css li:before{ content: '稍等-'; } html <ul> <li>列表1</li> <li>列表2</li> <li>列表3</li> <li>列表4</li> <li>列表5</li> </ul> ``` 运行之后的效果如下： ![](../../image/2014-03-28css3-3.jpg) 由此可以看出before还是很有用的，实际上before配合content属性还可以做出更酷的效果，这个后面会有单独的记录。 ---------------------------------- ###after选择器 after选择器和before选择器恰好相反一个是在后面一个是在前面，修改上面css如下： ``` css li:after{ content: '--说的'; } ``` 效果如下： ![](../../image/2014-03-28css3-4.jpg) ------------------------------ 上面就是四个伪元素选择器，最常用的是after和before选择器。后节会记录结构性伪类选择器。
